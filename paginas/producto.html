Ledger

Es un elemento que se utiliza para almacenar la información de acceso a las criptomonedas y la información de pago. Las wallets sirven para que tengamos la custodia de nuestras criptomonedas y tengamos acceso a ellas donde y cuando queramos. Una wallet permite crear una clave privada y a partir de esta, generar la clave pública y las direcciones. Estas carteras para criptomonedas pueden ser «hot wallet» o pueden ser «cold wallet», algo que explicaremos a continuación. Además, estas wallets pueden ser gestionados o protegidas por terceros, como las de las exchange.
